# Propagation Test Title Engine

These titles are for radar screening only. They are not final publishing titles.

Good radar titles should make the content angle obvious in one glance:

- "这个开源项目，把 XX 变成了 5 分钟的事"
- "一个小团队做了个很反常识的 AI 工具"
- "这篇论文不是刷榜，它可能会改变 XX 工作流"
- "别只看大模型发布，这个底层工具更值得盯"
- "这个 AI 产品的价值不在功能，在使用场景"

Avoid:

- "震惊"
- "全网首发"
- "彻底颠覆"
- "普通人必看"
- "AI 又杀疯了"

The hook can be sharp, but the core fact must stay honest.
